### Clash for Android 配置

1. **下载安装**
   - 从 GitHub Releases 下载 APK
   - 安装并打开应用

2. **导入配置**
   ```
   方式1: URL 导入
   - 点击 "配置" → "新配置" → "URL"
   - 粘贴订阅链接
   - 点击下载
   
   方式2: 文件导入
   - 将 YAML 配置文件放入手机
   - 点击 "配置" → "文件"
   - 选择配置文件
   ```

3. **启动代理**
   - 返回首页
   - 点击右上角开关
   - 允许 VPN 权限

### v2rayNG 配置

```
1. 打开 v2rayNG
2. 点击右上角 + 
3. 选择 "从剪贴板导入" 或手动添加
4. 选择节点，点击右下角 V 图标启动
```

### Shadowrocket 配置

1. **购买安装**
   - 使用非中国区 Apple ID
   - App Store 搜索 Shadowrocket
   - 购买并下载

2. **添加节点**
   ```
   方式1: 扫码导入
   - 点击右上角 +
   - 选择 "扫描二维码"
   
   方式2: 订阅导入
   - 点击右上角 +
   - 选择 "类型: Subscribe"
   - 粘贴订阅链接
   ```

3. **启用代理**
   - 选择节点
   - 打开首页开关
   - 允许 VPN 配置

## 🔧 常见问题

### Android

**Q: 为什么无法后台运行？**

**A:**
1. 关闭电池优化
2. 允许自启动
3. 锁定后台应用

**Q: 订阅更新失败？**

**A:**
1. 检查网络连接
2. 确认订阅链接有效
3. 尝试手动更新

### iOS

**Q: 如何购买 Shadowrocket？**

**A:**
1. 注册非中国区 Apple ID
2. 充值余额或绑定支付方式
3. App Store 购买

**Q: 为什么经常断线？**

**A:**
1. 开启 "始终开启 VPN"
2. 关闭 "按需连接"
3. 检查后台刷新设置
